Adding a Blur and Enlarging effect on Hover

If you want to add a nice effect on your images to make them pop and add some jazz to your page, these effects will be a nice addition. First we have the Enlarge effect which enlarges an image on a hover which is goos for smaller screens where you are trying to make the image bigger without compromising the layout of the page and even on normal size screens to help focus on the picture and even help the visitors see the picture better. Next is the blur effect, a really cool effect that blurs out everything but the picture you are hovering over. It adds emphasis on the picture you want to see and also helps the visitor focus on the image incase they have a hard time focusing on a single image when there are multiple nearby.

To start, let's begin with the enlarge on hover effect. First you want to find the image on the page you want to apply the effect to. For this example I will use
"<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">" In the div you want to add class so that you can target that specific image. " <div class="grow"> "
and put the img into the div class.
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
Then in your style.css create a line for the image. ".grow img { }" Inside the brackets you will write "transitions: 1000ms;" formated like so.
".grow img{
transition: 1000ms;
}"
This is setting the time it takes for the scaling to happen. 1000ms is 1second. Continuing in the css make a new line for the hover effect. The following ".grow img:hover { }" Then fill in the following the brackets with
".grow img:hover {
-webkit-transform: scale(1.4);
-ms-transform: scale(1.4);
transform: scale(1.4);
transition: 1000ms ease;
}"

These webkits are just for making sure that the hover effect works across all web browsers. The scale is what the transform is doing and inside the () is how much is will scale by" After this the hover effect should be applied and when you put your cursor over the image the photo should now enlarge.

Moving on to the next effect we are going to add a blur effect to everything but the image you hover over. First lets start by putting everything we just did in html into a div. We will call it blur.
<div class="blur">
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
</div>
And then we will create another div inside that div called card.
<div class="blur">
<div class="card">
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
</div>
