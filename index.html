Adding a Blur and Enlarging effect on Hover

If you want to add a nice effect on your images to make them pop and add some jazz to your page, the blur and enlarge effects can add visual interest and emphasis to images on a web page. They can help draw the user's attention to a specific image and make it stand out from the rest. Additionally, the blur effect can also help create a sense of focus and clarity by blurring out the surrounding content. This can be particularly useful when an image is the main focus of a page or section, and you want the user to be able to view it clearly. Overall, these effects can be a useful way to enhance the visual appeal of images on a web page and make them more engaging for the user. First we have the Enlarge effect which enlarges an image on a hover which is good for smaller screens where you are trying to make the image bigger without compromising the layout of the page and even on normal size screens to help focus on the picture and even help the visitors see the picture better. Next is the blur effect, a really cool effect that blurs out everything but the picture you are hovering over. It adds emphasis on the picture you want to see and also helps the visitor focus on the image incase they have a hard time focusing on a single image when there are multiple nearby.

To start, let's begin with the enlarge on hover effect. First you want to find the image on the page you want to apply the effect to. For this example I will use
"<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">" In the div you want to add class so that you can target that specific image. " <div class="grow"> "
and put the img into the div class.
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
Then in your style.css create a line for the image. ".grow img { }" Inside the brackets you will write "transitions: 1000ms;" formated like so.
".grow img{
transition: 1000ms;
}"
This is setting the time it takes for the scaling to happen. 1000ms is 1second. Continuing in the css make a new line for the hover effect. The following ".grow img:hover { }" Then fill in the following the brackets with
".grow img:hover {
-webkit-transform: scale(1.4);
-ms-transform: scale(1.4);
transform: scale(1.4);
transition: 1000ms ease;
}"
These webkits are used to target specific browsers and ensure that the styles are applied correctly. The scale property is used to specify how the element should be transformed, in this case by scaling it up by 1.4 times its original size. After this the hover effect should be applied and when you put your cursor over the image the photo should now enlarge.

Moving on to the next effect we are going to add a blur effect to everything but the image you hover over. First lets start by putting everything we just did in html into a div. We will call it blur.
<div class="blur">
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
</div>
And then we will create another div inside that div called card.
<div class="blur">
<div class="card">
<div class="grow">
<img src="img/shoes-640w.jpg" alt="a bunch of left shoes">
</div>
</div>
Next open the css file. Going to create a new line targeting the div class="blur"
".blur {
display: flex;
justify-content: space-evenly;
}"
Now create one targeting div class="card" by doing the following
".card {
padding: 10px;
margin: 0px 10px;
text-align: center;
transition: 500ms;
}"
This is adding some style to the div. Time to create the hover effect for the blur. Create a new line for the hover effect like so.
".blur:hover .card {
filter: blur(10px);
}"
This creates a blur effect when you hover over the image but it also blurs out the image, so let's fix that. Create a new line for this
".profile .card:hover {
filter: blur(0px);
}"
To prevent the image itself from being blurred, the .profile .card:hover selector is used to target .card elements that are within a .profile container that is being hovered over. When this happens, the filter property is applied with a blur value of 0px, which removes the blur effect from the image. This can be a useful way to create visual emphasis and focus on a specific image within a grid of images.
